修改配置项教程

代码中药自行改动apikey和userid的地方不多，除了技能里的

因为tts、asr都没有调用次数限制，所以这个问题不大

唯独技能部分，一些服务会做访问限制，所以建议自己换上appkey

还有的要改的就是访问城市的部分，城市要改为自己（使用者）的

网页中为了赶工，没有开发网页版配置，如果造成了一些麻烦，请谅解。所以你可以手动更改或在CLI中更改

更换方法：

1. 手动更改：直接更改config目录下的json文件即可，具体配置需要问插件的开发者

2. CLI更改：你可以通过“config list”命令来获得一个列表，这个列表里的东西是你所有可以更改的配置

通过”config get 配置名“ 来获取你的配置，注意：配置名里是没有”‘“、”“”、”.json“的！！！

然后你就获得了一个json格式的配置。修改你的数值就可以了。

最后，你可以”config set 配置名 json utf-8 配置json“来保存配置。主要：配置json里不能有任何的换行符，空格！！！！！！

感谢支持！！！

示例：

[RingRobotX] > config list
INFO cli.py 126 [CLI] 运行指令：config
INFO cli.py 128 [CLI] 指令返回：['api-version.json', 'ASR_BAIDU_API.json', 'Clock.json', 'func.json', 'RingRobotX_Web.json', 'TTS_BAIDU_API.json', 'Turing_RobotChat.json']

[RingRobotX] > config get Clock
INFO cli.py 126 [CLI] 运行指令：config
INFO cli.py 128 [CLI] 指令返回：{"set_time": "0", "playPath": ""}

[RingRobotX] > config set Clock json utf-8 {"set_time":"06:15","playPath":"/home/pi/hengaaaaaa.wav"}
INFO cli.py 126 [CLI] 运行指令：config
INFO cli.py 128 [CLI] 指令返回：OK
